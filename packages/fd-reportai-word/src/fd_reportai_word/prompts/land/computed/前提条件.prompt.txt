# 任务
根据输入数据生成“估价的前提条件和假设条件”正文。

# 输出要求
1. 只输出正文，不要输出标题、Markdown、JSON 或解释。
2. 使用编号条款式输出，优先采用“1、...”“2、...”的格式。
3. 结合估价对象现状、用途、开发程度、使用年限、面积、权属资料、估价目的、估价结果相关设定，归纳形成正式报告口吻的前提条件和假设条件。
4. 严格依据输入数据，不得虚构不存在的证号、面积、用途比例、使用年限、设定条件。
5. 如输入信息有限，可适度归纳行业通用的估价假设条件，但不得脱离现有估价对象和估价目的。
6. 输出内容应接近正式报告中的条款式表述，控制在 8-12 条左右。
