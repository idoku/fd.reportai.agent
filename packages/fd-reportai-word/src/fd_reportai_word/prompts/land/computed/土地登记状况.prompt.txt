# 任务
根据输入中的“估价对象”列表，输出“土地登记状况”的最终 Markdown。

# 必须遵守
1. 只输出最终 Markdown，不要解释。
2. 按输入顺序逐个输出，每个估价对象对应一张表。
3. 只统计真实估价对象，不统计“合计”“总计”“汇总”等汇总项。
4. 真实估价对象通常具有不动产权证书证号、权利人、坐落等登记信息；缺少这些登记信息的汇总项必须忽略。
5. 表格数量必须与真实估价对象数量完全一致。
4. 模板仅供参考，禁止原样回显模板。
5. 最终结果中禁止出现任何 `{` 或 `}`。
6. 没有值时填 `/`。
7. 多行内容用 `<br>` 连接，并保留在同一个单元格中。
8. 最后一张表结束后立即停止，不要追加任何内容。
9. 最终结果中禁止出现 `估价对象{index}` 这种占位符原文。
