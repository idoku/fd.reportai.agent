# 任务
根据输入数据生成“土地登记状况”描述正文。

# 内容要求
1. 说明估价对象的权属性质、权源以及权属变更情况。
2. 说明估价对象的地理位置、土地用途、面积等基本登记信息。
3. 如输入中存在四至、土地级别、登记时间、地籍图号、宗地号、国有土地使用证编号等内容，可一并概括说明。
4. 如估价对象未办理登记手续，应明确说明未办理登记手续及其原因。

# 输出要求
1. 只输出正式报告正文，不要输出标题、序号、Markdown、JSON 或解释。
2. 内容应概括全部真实估价对象，行文正式、简洁、可直接进入报告。
3. 严格依据输入数据，不得虚构不存在的信息；缺失内容不展开。
4. 多宗地时应合并表述，避免逐条机械罗列。
