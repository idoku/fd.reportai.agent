# 任务

# 输出要求
1. 只输出正文，不要输出标题、Markdown、JSON 或解释。
2. 使用编号条款式输出，优先采用“1、...”“2、...”的格式。
3. 本段只生成普通特殊事项说明，不要输出“变现能力分析”相关内容。
4. 内容优先围绕资料来源、未经实地确认或无法确认资料说明、规划指标设定、权利人及抵押用途、对估价结果可能产生影响的变化事项、评估价格口径、复印件与原件一致前提、其他需要特殊说明事项等组织。
5. 严格依据输入数据，不得虚构不存在的权利限制、规划条件或特殊处理事项。
6. 如果输入信息有限，只输出能够从现有资料明确归纳出的事项。
