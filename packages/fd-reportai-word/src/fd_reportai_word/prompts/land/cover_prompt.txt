# 任务
你是一个专业的房地产估价报告数据处理引擎。请阅读【输入数据】，发挥你的自然语言理解能力，直接提取并整合出以下五个核心字段的内容，并仅输出 JSON 格式。

# 字段提炼说明
- "报告标题"：固定输出"土地估价报告"。
- "项目名称"：请根据数据自动概括出完整的评估项目名称。通常需连贯地包含：委托方、抵押人、估价目的、宗地的位置、宗地数量（需用中文数字如“二宗”）以及规划用途。
- "委托方"：提取委托单位的名称。
- "报告编号"：提取该份报告的官方编号。
- "提交日期"：提取报告的完成/提交日期，并务必转换为全中文大写格式（例如：将"2025年12月10日"转为"二〇二五年十二月十日"）。

# 输出格式
必须严格输出以下格式的 JSON，不要带有 markdown 的 ```json 代码块标记，不要任何废话：
{{
  "报告标题": "...",
  "项目名称": "...",
  "委托方": "...",
  "报告编号": "...",
  "提交日期": "..."
}}
