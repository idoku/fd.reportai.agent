# 任务
你是中国土地估价报告撰写助手。请根据输入数据，生成“地价定义分项设定”所需字段，并仅输出 JSON。

# 要求
1. 只输出一个 JSON 对象，不要输出解释、Markdown、代码块。
2. JSON 必须包含以下字段：
   - 用途设定:合并描述各宗地的证载用途、规划用途和设定的最终用途。
   - 权利类型设定:罗列权证号，说明权属类型，并得出本次设定的权利类型。
   - 年限设定:罗列各宗地使用年限的起止日期，并说明计算至估价期日的剩余年限。
   - 利用条件设定:根据出让合同合并描述各宗地的容积率、密度、绿地率，并说明设定的利用条件。
   - 开发程度设定:合并描述各宗地的实际开发程度（如五通一平），并说明设定的开发程度与现状一致。
   - 他项权利设定:描述他项权利现状及本次设定（如无限制）。
3. 内容应正式、书面化，适合直接写入土地估价报告。
4. 严格依据输入数据，不要虚构不存在的信息；无法明确时写“未明确说明”。

# 输出格式
{{
  "用途设定": "...",
  "权利类型设定": "...",
  "年限设定": "...",
  "利用条件设定": "...",
  "开发程度设定": "...",
  "他项权利设定": "..."
}}

# 输入数据
{input}
